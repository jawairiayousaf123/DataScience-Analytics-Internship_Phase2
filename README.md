## 📂 Project Overview

### 1. 🛍️ Mall Customer Segmentation
*   **Objective:** Segment customers based on purchasing behavior to enable personalized marketing strategies.
*   **Techniques:** Unsupervised Learning (K-Means Clustering), PCA for dimensionality reduction, and Exploratory Data Analysis (EDA).
*   **Outcome:** Identified 5 distinct customer groups with actionable marketing strategies for each.

### 2. 📊 Executive Sales Dashboard
*   **Objective:** Develop an interactive Business Intelligence tool for global sales analysis.
*   **Techniques:** Data preprocessing, KPI calculation, and interactive visualization.
*   **Deployment:** Built using **Streamlit** with dynamic filters for region and category-based performance tracking.

### 3. 🏦 Loan Default Risk Prediction & Cost Optimization
*   **Objective:** Predict loan defaults and minimize financial loss using a cost-benefit optimization approach.
*   **Techniques:** Binary Classification (Logistic Regression), Feature Importance Analysis, and Cost-Function based Threshold Tuning.
*   **Outcome:** Achieved an AUC of 0.74 and successfully optimized the decision threshold to mitigate high-risk financial losses.
